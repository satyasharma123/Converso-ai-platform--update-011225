## âœ… **Key Architectural Principles**

1. **One Conversation Per Thread**
   - Gmail: One conversation per `threadId`
   - Outlook: One conversation per `conversationId`

2. **Message-Level Folders**
   - Folder stored in `messages.provider_folder`
   - NOT in `conversations.email_folder`

3. **Conversation Immutability**
   - `sender_name`, `sender_email`, `subject` set once
   - Never updated, even when new messages arrive

4. **Folder Derivation**
   - UI folder = latest message's `provider_folder`
   - Single query optimization (no N+1)

5. **Duplicate Prevention**
   - Unique constraint on `provider_message_id`
   - Check before insert

6. **Unified Sync Pipeline**
   - ALL folders use SAME sync logic
   - ONLY difference: provider label/folder name
