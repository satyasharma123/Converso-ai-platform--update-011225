## ğŸ” Troubleshooting

### "Cannot send message: Missing conversation details"
**Solution**: The conversation is missing required data. Try:
1. Refresh the page
2. Re-select the conversation
3. Check if LinkedIn account is connected

### "Your LinkedIn account is disconnected..."
**Solution**: Reconnect your LinkedIn account:
1. Go to Settings
2. Navigate to Connected Accounts
3. Click "Reconnect" on your LinkedIn account

### Attachment not uploading
**Solution**: Check:
1. File size is under 15MB
2. File type is supported
3. Internet connection is stable

### Message not appearing
**Solution**: 
1. Wait a few seconds for sync
2. Refresh the conversation
3. Check if message was actually sent (look for error toast)
